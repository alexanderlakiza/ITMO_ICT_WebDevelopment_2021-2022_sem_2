### Добавить данные о паре 

**URL** : `/pair/create/`

**HTTP 201 Created**

**Allow** : POST, OPTIONS

**Content-Type** : application/json

**Vary** : Accept

**JSON** :
```json
{
    "id": 1,
    "pair_number": 1,
    "name_day": "Mon",
    "room": 31,
    "group": 1,
    "teacher": 1,
    "subject": 1
}
```
