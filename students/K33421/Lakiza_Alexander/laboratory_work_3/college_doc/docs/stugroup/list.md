### Просмотреть данные о группах студентов 

**URL** : `/stugroup/list/`

**HTTP 200 OK**

**Allow** : GET, HEAD, OPTIONS

**Content-Type** : application/json

**Vary** : Accept

**JSON** :
```json

[
    {
        "id": 1,
        "student": 1,
        "group": 1
    },
    {
        "id": 2,
```
